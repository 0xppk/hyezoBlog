### 테스트
> 탭 언더라인 슬라이딩 UI
- 탭을 누를 때 아래밑줄이 미끄러지며 따라오는 효과를 지칭합니다.
- 기존에 class 탈부착으로만 구현해봐서 조금 더듬거렸어요.
- props를 이용하여 간단히 구현 했습니다. 일반적인 레이아웃 잡는 데서보다, 이렇듯 애니메이션을 주는 데서 동적 스타일링의 매력이 느껴지는 듯했어요.
```js ---.tsx
const Lines = styled.div<styleProps>`
  flex-basis: 33.333333%;
  background-color: blue;
  transform: translateX(${(props) => props.x}%);
  transition: transform 0.6s;
`;
```
