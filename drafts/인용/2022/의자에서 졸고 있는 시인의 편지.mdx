**〈의자에서 졸고 있는 시인의 편지〉**  
<br/>
**편지1**  
나는 사물을 있는 그대로 말한다  
우리는 모든 것을 알고 있거나  
아무것도 절대로 아무것도 몰라.  
<br/>
우리에게 허락된 유일한 것은  
정확하게 말하는 연금술을 배우는 것.  
<br/>

**편지2**  
밤새 내내 나는 여자들 꿈을 꾸지  
어떤 여자들은 분명한 모습으로 나를 비웃고  
어떤 여자들은 요염한 모습을 보여 주지  
여자들과의 전쟁은 끝없이 전개되고  
편안한 때가 한시도 없어  
<br/>
그러다 벼락맞은 얼굴로 잠자리에서 일어나면  
나는 두려움에 질려 미칠 것만 같아.  
<br/>

**편지3**  
죽음에 대해서는 아무 말도 하지 않은 채  
자신의 피조물들을 자유의지라는 이름의  
운명에 맡기고 늙어감과 질병이라는  
은혜에 내버려두고 신을 믿는 것은  
무척 힘이 드는 작업이다.  
<br/>

**편지4**  
나는 호화마차를 보면 절을 하고 싶은 부류에 속한다.  
<br/>

**편지5**  
젊은이들이여  
하고 싶은 대로 글을 쓰라  
너희에게 가장 좋아 보이는 문체로.  
이 몸이 생각컨대, 단 하나의  
길만이 진리라는 믿음 때문에  
우리는 그 동안 지나치게 많은 피를  
다리 밑으로 흘려 오지 않았는가.  
시에서는 모든 것이 허용된다.  
<br/>

**편지6**  
병들림과  
<span className="opacity-0">-----</span>늙어감  
<span className="opacity-0">---------</span>죽음이  
순진한 아가씨처럼 춤을 춘다  
백조의 호수 주위를 맴돌며  
반쯤 벗은 몸과  
<span className="opacity-0">----------</span>취기 어린 눈동자와  
산호빛 입술에 음란함을 머금은 채.  
<br/>

**편지7**  
달에 아무도 살지 않음은 명백한 사실이다  
<br/>
의자가 탁자인 것처럼  
끝없는 움직임으로 나비가 꽃이 되는 것처럼  
진리가 집단적 오류인 것처럼  
영혼이 죽음과 함께 생명을 잃는 것처럼  
<br/>
주름이 흉터가 아님은  
명백한 사실이다.  
<br/>

**편지8**  
이런저런 이유로 나의 작은  
율법의 탑으로부터  
내려와야 할 때마다  
나는 추위로 오들오들 떨며 돌아오곤 한다  
고독과  
<span className="opacity-0">------</span>두려움과  
<span className="opacity-0">-----------</span>고통으로  
몸부림치며.  
<br/>

**편지9**  
전차는 이미 떠났고  
나무들은 모두 잘렸다  
지평선은 십자가로 가득 찬다.  
<br/>
막스는 일곱 번 거부당했고  
우리는 여전히 그곳을 쫓아간다.  
<br/>

**편지10**  
벌들에게 쓸개 먹이기  
아가리에 정액을 접종하기  
피웅덩이에서 무릎꿇기  
신성한 예배당에서 재채기하기  
소의 젖을 짜서  
소의 머리 위에 부어 버리기.  
<br/>

**편지11**  
아침식사 시간의 검은 뭉게구름과  
점심식사 시간의 천둥을 지나  
저녁식사 시간의 번개에 이르기까지.  
<br/>

**편지12**  
남들에게 진지하게 보이기 위해  
짐짓 슬픈 표정을 짓는 것은 싫다.  
내게는 두개골조차도 웃음이 된다.   
십자가에서 잠이 든 시인은  
피눈물을 뿌리며 사람들에게 인사를 한다.  
<br/>
– 니카노르 파라, 『벽에 그려진 얼굴들』 중에서 (전기순 역)
