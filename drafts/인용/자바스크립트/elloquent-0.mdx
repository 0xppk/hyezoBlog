자바스크립트에 대한 나의 이해 자가진단.. 치부를 드러내자ㄷㄷ 

### 1. 무엇이 출력될까? (호이스팅)
```js
function  sayHi(){
  console.log(name);
  console.log(age);
  var name = "Lydia";
  let age = 21;
}

sayHi();
```
<br/>
_`답`_ : `undefined`, `ReferenceError`  
_`이유`_ : 변수 `var`는 선언과 동시에 scope의 최상단으로 호이스팅된다. 이때 초기화도 같이 이루어지며 기본값은 `undefined`다. 후에 할당값이 붙게 된다.  
마찬가지로 `let`도 호이스팅된다. 하지만 `var`와 달리 초기화가 자동으로 되지 않기 때문에 할당이 이루어지기 전까진 값에 접근할 수 없다.  
_`사족`_ : 변수처럼 함수도 호이스팅된다. 함수의 기본 형태인 `선언식`들의 경우 순서에 상관없이 서로를 참조/호출한다. 이런 함수들을 변수에 바인딩한 것이 함수 `표현식`인데, 그 경우 변수 호이스팅 원칙을 적용받아 순서에 엄격을 가할 수 있을 것이다.
