##### 🌊❗️Notice❗️🌊
```zsh 설명서
Tailwind는 공홈 설명이 정말 잘돼있다.
그래서 클래스마다 어떤 css가 구현된 건지
기회삼아 찬찬히 살펴보며 공부했다.
아무리 편리함의 노예라지만 실력은 늘려야 하니까.

만약 VSCode 사용자라면 확장프로그램에서
> Tailwind CSS IntelliSense 를 반드시 설치하자.
자동완성 기능을 제공해주는데(나는 컨트롤+스페이스를 이용한다),
클래스명이 정확히 기억나지 않을 때 많은 도움이 된다.

쓰면서 어느 정도 적응이 되었다면 커스터마이징하면 된다.
아래 설정은 내가 이 블로그에 쓰고 있는 예시인데,
keyframes, clamp, 웹폰트 등 쓰고 싶은 건 다 쓸 수 있다.

어떤 프로퍼티는 extend, 어떤 건 theme,
또 어떤 건 utilities로 아예 새로 파줘야 하니 
자세한 사항은 그때그때 공홈을 찾아 읽어보길 바람
```

##### config 설정
```js /tailwind.config.js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/pages/**/*.{js,ts,jsx,tsx}",
    "./src/components/**/*.{js,ts,jsx,tsx}",
    "./drafts/**/*.{md,mdx}",
  ],
  darkMode: "class",
  theme: {
    extend: {
      transitionDuration: {
        3000: "3000ms",
      },
      keyframes: {
        wiggle: {
          "0%, 100%": { transform: "rotate(-3deg)" },
          "50%": { transform: "rotate(3deg)" },
        },
        swap: {
          "50%": { transform: "translateY(50px)" },
          "100%": { transform: "translateY(-250px)", opacity: 0 },
        },
        swapReverse: {
          "0%": { transform: "translateY(150px)", opacity: 0 },
          "100%": { transform: "translateY(0px)", opacity: 1 },
        },
      },
      animation: {
        wiggle: "wiggle 1s ease-in-out infinite",
        swap: "swap 0.8s ease-in-out",
        swapReverse: "swapReverse 0.8s ease-in-out",
      },
      colors: {
        waterMelon: "#ED4C6A",
      },
    },
    screens: {
      sm: "375px",
      md: "768px",
      lg: "1024px",
      xl: "1440px",
    },
    fontSize: {
      sm: "clamp(0.67rem, calc(0.59rem + 0.22vw), 0.76rem);",
      base: "clamp(0.81rem, calc(0.7rem + 0.36vw), 1rem);",
      md: "clamp(0.84rem, calc(0.73rem + 0.39vw), 1.08rem);",
      lg: "clamp(1.17rem, calc(0.95rem + 0.88vw), 1.76rem)",
      xl: "clamp(1.40rem, calc(1.09rem + 1.08vw), 1.95rem)",
      "2xl": "clamp(1.69rem, calc(1.25rem + 1.48vw), 2.44rem)",
      "3xl": "clamp(1.9rem, calc(1.44rem + 1.67vw), 2.84rem)",
      mono: "clamp(0.76rem, calc(0.63rem + 0.26vw), 0.85rem);",
    },
    fontFamily: {
      sans: ["ui-sans-serif", "system-ui"],
      serif: ["ui-serif", "Georgia"],
      mono: ["IBMPlexMono", "ui-monospace", "SFMono-Regular"],
      custom: ["IBMPlexSans", "NanumSquareR"],
    },
    minHeight: {
      content: "100vh",
    },
    listStyleType: {
      none: "none",
      disc: "disc",
      decimal: "decimal",
      square: "square",
      circle: "circle",
      roman: "upper-roman",
    },
  },
  plugins: [],
};
```

##### css 설정
```css /src/styles/tailwind.css
@tailwind base;
@layer base {
  *::-webkit-scrollbar {
    display: none;
  }
  html,
  body {
    @apply h-full scroll-p-8 scroll-smooth font-custom;
  }
  /* blockquote {
    @apply text-emerald-600;
  } */
  a {
    @apply underline decoration-rose-400/80 decoration-solid underline-offset-8 hover:decoration-wavy;
  }
  h1 {
    @apply text-3xl;
  }
  h2 {
    @apply mb-5 text-2xl;
  }
  h3 {
    @apply py-4 text-lg font-bold text-waterMelon dark:text-cyan-200/90;
  }
  h4 {
    @apply pt-4 pb-2 text-md font-bold text-emerald-600;
  }
  h5 {
    @apply text-base;
  }
  ul {
    @apply list-disc pt-1 pl-6;
  }
  ol {
    @apply list-decimal pt-1 pl-6;
  }
}

@tailwind components;
@layer components {
  #__next {
    @apply h-full w-full;
  }
  .wrap {
    @apply flex items-center justify-center;
  }
}

@tailwind utilities;
@layer utilities {
  .keep-all {
    word-break: keep-all;
  }
  .align-text-20 {
    vertical-align: -20%;
  }
}
```
