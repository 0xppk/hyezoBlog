간단하게 정리하는 Tailwind.  
인스톨은 앞의 세단계, 커스터마이징은 뒤의 세단계로 나눠서 정리했다.  
공홈이 진짜 잘돼있으니까 제발 <Lnk text={'공홈'} href={'https://tailwindcss.com/'} />을 봐주세요..!!


<CH.Spotlight rows={20}>
```js terminal
// yarn
yarn add -D tailwindcss postcss autoprefixer

// npm
npm install -D tailwindcss postcss autoprefixer
```

##### 초기 설정
<CH.Code>
```js /tailwind.config.js
/** @type {import('tailwindcss').Config} */ 
module.exports = {
  content: [
    "./pages/**/*.{js,ts,jsx,tsx}",
    "./components/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```
---
```js /postcss.config.js
module.exports = {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
};
```

##### config 설정
```js /tailwind.config.js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/pages/**/*.{js,ts,jsx,tsx}",
    "./src/components/**/*.{js,ts,jsx,tsx}",
    "./drafts/**/*.{md,mdx}",
  ],
  darkMode: "class",
  theme: {
    extend: {
      transitionDuration: {
        3000: "3000ms",
      },
      keyframes: {
        wiggle: {
          "0%, 100%": { transform: "rotate(-3deg)" },
          "50%": { transform: "rotate(3deg)" },
        },
        swap: {
          "50%": { transform: "translateY(50px)" },
          "100%": { transform: "translateY(-250px)", opacity: 0 },
        },
        swapReverse: {
          "0%": { transform: "translateY(150px)", opacity: 0 },
```

##### css 설정
```css /src/styles/tailwind.css
@tailwind base;
@layer base {
  *::-webkit-scrollbar {
    display: none;
  }
  html,
  body {
    @apply h-full scroll-p-8 scroll-smooth font-custom;
  }
  /* blockquote {
    @apply text-emerald-600;
  } */
  a {
    @apply underline decoration-rose-400/80 decoration-solid underline-offset-8 hover:decoration-wavy;
  }
  h1 {
    @apply text-3xl;
  }
  h2 {
    @apply mb-5 text-2xl;
  }
  h3 {
    @apply py-4 text-lg font-bold text-waterMelon dark:text-cyan-200/90;
  }
  h4 {
    @apply pt-4 pb-2 text-md font-bold text-emerald-600;
  }
  h5 {
    @apply text-base;
  }
  ul {
    @apply list-disc pt-1 pl-6;
  }
  ol {
    @apply list-decimal pt-1 pl-6;
  }
}

@tailwind components;
@layer components {
  #__next {
    @apply h-full w-full;
  }
  .wrap {
    @apply flex items-center justify-center;
  }
}

@tailwind utilities;
@layer utilities {
  .keep-all {
    word-break: keep-all;
  }
  .align-text-20 {
    vertical-align: -20%;
  }
}
```

```js .prettierrc.js
  // yarn berry에선 플러그인 위치를 특정해줘야 함(원래는 자동)

  tabWidth: 2,
  printWidth: 90,
  trailingComma: "all",
  bracketSameLine: false, 
  arrowParens: "avoid",
};
```
